# Manong Novel

AI 小说写作桌面客户端 — 写作台、灵感模式与模型服务一体化。

## 0.4.0

- **新手引导**：首次登录后可选情境式引导（示例作品 + Coachmark），设置中可重新体验
- **导入增强**：导入管线重构，实体抽取、蓝图 JSON、覆盖度与字段清洗，导入状态与质量反馈更清晰
- **阅读体验**：阅读分页与导航优化，独立阅读窗更稳定
- **语言设置**：设置内独立语言面板，中英文切换更直观
- **灵感模式**：建议芯片与立项对话体验优化
- **工程化**：网关错误格式化、导入/引导相关单测补充，Agent 编排与资源锁加固

## 0.3.0

- **写作管线重构**：章节生成迁移至主进程，写作服务拆分为模块化子系统，提升稳定性与可测试性
- **Agent 编排**：新增多 Agent 工作流、资源锁与编排器，支持复杂生成任务协调
- **故事系统**：概念 / 蓝图 / 章节提交记录与版本历史，故事一致性检查（Story Doctor）
- **国际化**：内置中英文界面切换（zh-CN / en-US）
- **作品详情增强**：Agent 日志、故事提交历史、通用文本编辑弹窗，蓝图编辑流程简化
- **灵感模式**：重构对话式立项与蓝图生成流程
- **章节质量**：章节大纲质量检查、陈旧章节自动恢复
- **工程化**：引入 ESLint，CI 新增 lint 门禁；Release workflow 完善 Windows / macOS 双平台构建

## 0.2.0

- Novel 品牌重构：Arena 命名统一为 Novel，UI 与样式体系对齐
- 素材库：角色概览面板与编辑体验优化
- 文档：新增开发文档与界面预览说明

## 0.1.0

- 写作台：项目列表、新建、删除
- 灵感模式：快速创建项目
- 作品详情与写作台章节编辑
- 账户登录、钱包充值、API Key 与用量统计
- 模型请求经账户网关，Key 仅存本机
- 本地 JSON 持久化，无需 Python 后端
