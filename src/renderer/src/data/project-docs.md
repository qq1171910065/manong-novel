# Arboris Novel 使用说明

Arboris Novel 是一款 AI 小说写作桌面应用。你可以在写作台管理项目，用灵感模式快速立项，并通过模型服务辅助大纲、章节与设定生成。

## 快速开始

1. 登录 Arboris Novel 账户并完成邮箱验证。
2. 在「写作台」查看已有项目，或进入「灵感模式」创建新项目。
3. 进入作品详情维护蓝图、人物、伏笔等设定，再在写作台撰写与润色章节。
4. 在「账户设置 → 模型概览」确认网关连接与本机 API Key。

## 写作台

- 展示本地保存的全部小说项目。
- 支持新建、进入、删除项目。
- 可从 `.txt` 文件导入已有正文（若功能已启用）。

## 灵感模式

- 用自然语言描述创作意图，快速生成项目骨架。
- 适合从零起步或探索新题材。

## 模型服务

模型调用经账户后台配置的 New API 网关：

- 需在「API Key 管理」中创建并保存本机 Key。
- 钱包余额不足时，请在「钱包充值」中充值。
- 默认对话模型可在「模型概览 → 兜底对话模型」中设置。

若模型无法回复，请检查：账户服务是否在线、邮箱是否已验证、本机 Key 是否存在、钱包余额是否充足。

## 账户与设置

- **账户概览**：个人信息、网关状态、密钥数量。
- **钱包充值 / API Key**：模型计费与调用凭证。
- **用量明细**：查看近期模型调用记录。
- **显示与界面 / 声音**：调整界面缩放、动效与音效。
- **帮助中心**：基于本文档的智能问答。
- **问题反馈**：提交 Bug 与建议。

## 数据存储

小说项目保存在本机用户数据目录（按账户隔离），无需单独启动后端服务。

## 常见问题

**提示账户信息未就绪？**  
请退出后重新登录；若仍失败，在设置中检查 API 服务地址是否正确。

**写作台加载失败？**  
点击「重试」；确认已登录且网络可访问账户服务。

**模型网关不可用？**  
打开「模型概览」刷新连接状态，并确认管理员已启用对应模型渠道。
